Skripta vodi korisnika kroz proces odabira tipa izbora, izbornog kruga, regiona, opštine i biračkih mesta, zatim učitava rezultate glasanja i snima ih u CSV fajlove.

Testirano na macOS.

Neophodni programi: `jq`, `curl`, `sed`, `grep`.
